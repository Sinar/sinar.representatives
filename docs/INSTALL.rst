collective.popit Installation
-----------------------------

To install collective.popit using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``collective.popit`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        collective.popit

* Re-run buildout, e.g. with:

    $ ./bin/buildout
